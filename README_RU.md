# JADE
Плагин для League of Legends, который работает через [Pengu Loader](https://github.com/PenguLoader/PenguLoader/releases/).
---
JADE позволяет вам кастомизировать клиент League of Legends на ваш вкус.




______________________________________________________________
______________________________________________________________



<h1> Доступен следующий функционал: </h1>
<h2> 1. Фон </h2>
**
Заменяет фон в лобби, профиле призывателя, ТФТ и так далее.
**

Для его активации вам необходимо активировать нужный пункт в меню настроек клиента.

<img width="772" height="596" alt="image" src="https://github.com/user-attachments/assets/c6e957b8-bd30-4b18-a865-b03ee11068f8" />


Фон настраивается с помощью кнопки шестеренки, которая расположена рядом со значком призывателя.
