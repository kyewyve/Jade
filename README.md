# Jade
A comprehensive plug-in for the League of Legends client running through the [Pengu Loader](https://github.com/PenguLoader/PenguLoader/releases/).
---
Jade allows you to customize the League of Legends client at your discretion.

The following functionality is currently available:

<h1> 1. Background Changer </h1>
**
Replaces the background in the lol lobby, summoner's profile, TFT, and so on.
**

To activate the functionality, you need to install the necessary checkbox.

<img width="772" height="596" alt="image" src="https://github.com/user-attachments/assets/c6e957b8-bd30-4b18-a865-b03ee11068f8" />


The background is adjusted via the gear button, which is located next to the summoner icon.
